Authentication
- Determines whether users are who they claim to be
- Challenges the user to validate credentials (for example, through passwords, answers to security questions, or facial recognition)
- Usually done before authorization
- It isuaully needs the user's login details
- Generally, transmits info through an ID Token


## Methods
- Knowledge-based authentication
- Ownership-based authentication
- Inherence-based authentication

**Knowledge
- Passwords
- Passphrases
- PINs
- Answers to security questions

***Ownership
- ID card
- Security token
- Smartphone for authentication apps

**Inherence
- Fingerprint
- Facial patterns
- Voice recognition
- Signatures
