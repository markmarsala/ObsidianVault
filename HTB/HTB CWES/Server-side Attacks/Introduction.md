- Server-Side Request Forgery (SSRF)
- Server-Side Template Injection (SSTI)
- Server-Side Includes Injection (SSI)
- eXtensible Stylesheet Language Tranformation (XSLT) Server-Side Injection

SSRF: server sends requests to other servers
SSTI: inject template code
SSI: inject commands into the SSI directives
XSLT: manipulate XSLT transformations performed on the server. XSLT converts XML to other formats.
