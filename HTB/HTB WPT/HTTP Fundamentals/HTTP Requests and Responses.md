## HTTP Request

1. Method (GET, PUT, POST, etc.)
2. Path (/users/login.html)
3. Version (HTTP/1.1)
4. HTTP Headers (Host, User-Agent, Accept, etc.)


## HTTP Response

1. HTTP version (HTTP/1.1)
2. Response code (200 OK)
3. Response headers (Date, Server, Set-Cookie, etc.)
4. Response body (html code)
