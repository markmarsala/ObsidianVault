**Status Codes
1.. = provides information and does not affect the processing of the request
2.. = returned when a request succeeds
3.. = returned when the server redirects the client
4.. = signifies improper requests from the client. Requesting a resource that doesn't exist or requesting a bad format
5.. = returned when there is some problem with the HTTP server itself

**Common codes
200 OK
302 Found
400 Bad request
403 Forbidden
404 Not Found
500 Internal Server Error

All status codes: https://developer.mozilla.org/en-US/docs/Web/HTTP/Status
